# Customer Churn Predictive Analytics

## Project Overview
This project analyzes customer churn behavior for a SaaS company using predictive analytics and machine learning techniques. The objective was to identify high-risk customers, evaluate churn drivers, and develop proactive retention strategies.

---

## Business Problem
CloudCore, a mid-sized SaaS company, experienced increasing customer churn within its basic and mid-tier subscription plans. The project focuses on identifying patterns associated with churn and developing an early-warning analytics framework.

---

## Analytics Methods
- Exploratory Data Analysis (EDA)
- Logistic Regression
- Decision Trees
- Random Forest
- Gradient Boosting
- SMOTE (class balancing)
- Classification model evaluation
- ROC-AUC analysis

---

## Key Findings
- Month-to-month contracts showed significantly higher churn rates.
- Electronic check payment methods were associated with elevated churn risk.
- Tenure and contract structure were among the strongest predictive variables.
- Ensemble models improved predictive performance over individual decision trees.
